Compiling TypeScript to ES6 syntax

TypeScript 1.5 announced support for the import syntax of ES6. As it still in beta, you'll need to explicitely install it using NPM 

        npm install typescript@1.5.0-beta

That's basically all you need to get started. Now you can use the new import syntax. 

        import name from 'package';

Unfortunately, the browsers can't work yet with this new module loading system.
You could use a ES6 transpiler such as Traceur or Babel to rewrite the ES6 module syntax into a format that can be understood by the browser (albeit with help of an external library). There are a number of formats available:

  • AMD
  • CommonJS
  • System.js
  • UMD
However we could also choose to use System.js to allow doing this transpilation on the fly.

System.js allows us to choose one of thee possible transpilers:

  • Traceur
  • Babel
  • TypeScript
